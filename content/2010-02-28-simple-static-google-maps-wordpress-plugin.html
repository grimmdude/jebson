--- title:Simple Static Google Maps Wordpress Plugin description: keywords: comments: on ---

Write a lot of bar or restaurant reviews or about anything that has an address? This WordPress plugin will allow you to easily place a quick loading google map at the end of any post or page when desired.

Download Plugin

Usage

To use simply enter an address in the Static Map meta box found on the right side of the post/page editing screen. Note that the map will only show if something is entered in the Address field of the box. See live example at the bottom of this post.
