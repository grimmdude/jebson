Designs by Me

WordPress Plugins

Simple Popup Plugin

Use this plugin to create links to simple popup windows. Supports multiple popup links on posts/pages/widgets and window positioning/centering options.

Simple Auto Delay Popup Plugin

Simple Auto Delay Popup is a lightweight WordPress plugin for easily creating modal popup windows that automatically display after a certain amount of time, defined by you. Popup content can be the content of a page on your site or any other url. Usage is very straight forward and it’s very scalable.

Simple Static Google Maps

This plugin simply inserts a quick loading static google map at the bottom of a post or page when an address is entered in the custom field static_map_address.

Simple Select All Text Box

A plugin that makes it easy to create text boxes which automatically select everything inside with one click. It’s useful for song lyrics, directions, small bits of code, etc.

Web Applications

Name That Chord

An online guitar chord finder & identifier using jQuery, Underscore, and Raphael JavaScript libraries.

Name That Key

Another online tool similar to ‘Name That Chord’ but geared more for general use. It’s function is to show you the probable key of a song by the chords it uses. You can select the chords and instantly see the key highlighted in the circle of fifths (drawn strictly with CSS3 when viewed with a modern browser).

Scale Generator

A single page web application that generates scales on the fly every time you select a key by using the correct intervals for that scale type. Currently generates major, natural minor, or harmonic minor scale types in any key along with the pentatonic scale, diatonic triad notes and names, and relative key.

Other Cool Stuff

Raphael Musical Instruments

A guitar fretboard and piano keyboard drawn dynamically with vector graphics via the Raphael JavaScript library.

Flipping jQuery Menus

A scalable jQuery script which adds a ‘flipping’ effect to text based navigational menu items when hovered over (without needing background images).
